## 📂 项目列表

### 1. 原神合成台测试用例
**文件**：`Genshin_Crafting_TestCases.xlsx`

针对《原神》合成台系统设计的10+条测试用例，覆盖正常合成、材料不足、数量边界、背包满等场景

**测试场景覆盖**：
- 不同星级材料合成
- 单次合成1个、多个、最大数量（999）
- 材料不足时合成
- 快速合成功能
- 取消合成操作
- 背包空间不足
- 不同城市合成台功能验证

### 2. 原神角色配置表检查工具
**文件**：
- `check_character_data.py` （核心脚本）
- `check_report.xlsx` （示例输出报告）
用Python + openpyxl编写的自动化检查工具，用于检查游戏配置表中的数据异常。

**检查规则**：
- 角色ID重复检测
- 基础生命/攻击/防御数值范围检查
- 空值检测
- 星级合法性检查（仅允许4、5星）

**运行方式**：将`角色基础属性.xlsx`与脚本放在同一目录，执行 `python config_checker.py`，自动生成`check_report.xlsx`。
