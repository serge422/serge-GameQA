# 原神角色配置表检查工具

## 目的
游戏开发中，策划配置的角色数据可能有误（重复ID、数值越界等）。本工具自动检查这些错误，减少人工核对时间。

## 技术栈
- Python 3
- openpyxl

## 使用方法
1. 将需要检查的Excel文件（命名为“角色基础属性.xlsx”）与脚本放在同一目录。
2. 执行 `python check_character_data.py`。
3. 打开生成的 `check_report.xlsx` 查看异常。

## 学习收获
- 熟悉openpyxl库的基本操作
- 理解游戏测试中配置检查的常见规则
- 学会用AI辅助编写工具脚本并调试
