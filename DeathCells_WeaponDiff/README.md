# 死亡细胞武器数值版本对比工具

## 目的
对比死亡细胞两个版本（v3.3 vs v3.4）的武器属性表，自动找出新增、删除、数值变化的武器，用于版本更新前的平衡性调整验证。

## 技术栈
- Python 3
- pandas

## 使用方法
1. 将 `武器数据_v3.3.xlsx` 和 `武器数据_v3.4.xlsx` 放在同一目录。
2. 执行 `python weapon_diff.py`。
3. 打开生成的 `diff_weapon.xlsx` 查看差异。

## 学习收获
- 熟悉pandas的DataFrame操作与对比逻辑
- 理解版本管理中配置表对比的实际应用场景
- 增强使用AI辅助开发复杂脚本的能力
